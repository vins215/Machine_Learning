### Creating conda Virtual Envirornament
```
 conda create -p venv python==3.7 -y
 ```

 ```
 conda activate venv/
 ```


 ```
 pip install -r requirement.txt
 ```

 ```
 git add <file_name>
 ```

NOTE: To ignore file or the folder from git we can write the name of file/folder in .gitignore

git status
